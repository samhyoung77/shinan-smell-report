# shinanAPT

아파트 개울가 냄새 신고 모바일 웹앱. 아파트 주민들이 익명으로 악취 발생을 신고하고, 단지 간 발생 패턴을 통계로 비교하기 위한 정적 웹앱.

- **프론트엔드:** 정적 HTML/CSS/Vanilla JS, 빌드 도구 없음
- **백엔드:** Firebase Firestore (완전 익명, 로그인 없음)
- **배포:** GitHub Pages (공유 URL)

---

## 하네스: 냄새 신고 웹앱

**목표:** 디자인 → 프론트엔드 → Firestore 연동 → QA 순서의 파이프라인으로 이 웹앱을 구축·유지보수한다.

**트리거:** 이 프로젝트의 웹앱 관련 작업(초기 구축, 기능 추가/수정, 디자인 변경, 통계 개선, 배포 준비, 부분 재실행) 요청 시 `smell-report-orchestrator` 스킬을 사용하라. 단순 질문이나 파일 확인은 직접 응답 가능.
